Suppression de compte Kaba

Cette page explique comment supprimer votre compte Kaba et les données associées.

Comment supprimer votre compte

Méthode 1 : Via l'application (Recommandé)
  1. Ouvrez l'application Kaba
  2. Allez dans Profil → Paramètres
  3. Cliquez sur "Supprimer mon compte"
  4. Confirmez la suppression
Méthode 2 : Par email

Envoyez un email à ive.tawasol@gmail.com avec :

  • Objet : "Demande de suppression de compte Kaba"
  • Votre email ou numéro de téléphone associé au compte
  • Confirmation que vous souhaitez supprimer définitivement votre compte

Nous traiterons votre demande sous 30 jours maximum.

Que se passe-t-il après la suppression ?

⚠️ Attention : Cette action est irréversible

Données supprimées immédiatement :

  • Informations personnelles : Nom, prénom, email, numéro de téléphone
  • Photo de profil
  • Annonces actives (désactivées et supprimées)
  • Messages de chat
  • Historique du wallet
  • Notifications

Données conservées temporairement (30 jours) :

  • Transactions en cours (pour résolution de litiges)
  • Historique des signalements (pour modération)

Ces données sont automatiquement supprimées après 30 jours.

Données conservées de façon permanente (anonymisées) :

  • Statistiques générales (nombre de transactions, sans lien vers vous)
  • Données légales obligatoires (si requis par la loi)

Suppression partielle des données

Vous pouvez supprimer certaines données sans supprimer votre compte :

  • Annonces : Supprimez vos annonces individuellement dans "Mes annonces"
  • Messages : Supprimez des conversations dans le chat
  • Photo de profil : Supprimez-la dans "Modifier le profil"

Ces suppressions sont immédiates et définitives.

Avant de supprimer votre compte

Assurez-vous de :

  • ✅ Récupérer votre solde wallet (retirez vos fonds avant suppression)
  • ✅ Terminer toutes les transactions en cours
  • ✅ Sauvegarder les informations importantes (conversations, contacts)
  • ✅ Annuler les annonces actives

Réactivation du compte

Une fois votre compte supprimé, vous avez 30 jours pour changer d'avis :

  • Contactez-nous à ive.tawasol@gmail.com
  • Nous pourrons restaurer votre compte avec toutes vos données

Après 30 jours, la suppression est définitive et irréversible.

Besoin d'aide ?

Pour toute question concernant la suppression de votre compte :

  • Email : ive.tawasol@gmail.com
  • Réponse : Sous 48 heures
