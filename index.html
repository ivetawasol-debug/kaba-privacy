Politique de Confidentialité

Dernière mise à jour : 26 décembre 2024

1. Introduction

Kaba ("nous", "notre", "nos") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles lors de l'utilisation de notre application mobile.

2. Informations que nous collectons

Nous collectons les informations suivantes :

  • Informations d'identification : Nom, prénom, adresse email, numéro de téléphone
  • Informations de profil : Photo de profil, biographie
  • Informations de transaction : Historique des annonces, achats, ventes
  • Communications : Messages échangés via le chat de l'application
  • Localisation : Localisation approximative pour afficher les annonces pertinentes (avec votre consentement)
  • Photos et fichiers : Images uploadées pour les annonces
  • Informations financières : Solde du wallet, historique des transactions (aucune coordonnée bancaire n'est stockée)

3. Comment nous utilisons vos informations

Nous utilisons vos informations pour :

  • Créer et gérer votre compte utilisateur
  • Faciliter les transactions entre acheteurs et voyageurs
  • Permettre la communication via le chat intégré
  • Traiter les paiements et gérer le wallet
  • Améliorer nos services et développer de nouvelles fonctionnalités
  • Vous envoyer des notifications importantes concernant votre compte
  • Assurer la sécurité et prévenir la fraude
  • Respecter nos obligations légales

4. Partage de vos informations

Nous ne vendons jamais vos données personnelles. Nous partageons vos informations uniquement dans les cas suivants :

  • Avec d'autres utilisateurs : Votre nom, photo de profil et annonces sont visibles publiquement sur la plateforme
  • Fournisseurs de services : Services d'hébergement (Supabase), notifications (Firebase Cloud Messaging)
  • Obligations légales : Si requis par la loi ou pour protéger nos droits

5. Sécurité de vos données

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données :

  • Chiffrement des données en transit (HTTPS/SSL)
  • Stockage sécurisé sur des serveurs protégés
  • Accès restreint aux données personnelles
  • Authentification sécurisée avec vérification

6. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants concernant vos données personnelles :

  • Droit d'accès : Consulter les données que nous détenons sur vous
  • Droit de rectification : Corriger vos données inexactes
  • Droit à l'effacement : Supprimer votre compte et vos données
  • Droit à la portabilité : Recevoir une copie de vos données dans un format structuré
  • Droit d'opposition : Vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : privacy@kaba.app

7. Conservation des données

Nous conservons vos données personnelles tant que votre compte est actif. Si vous supprimez votre compte, vos données personnelles sont supprimées dans un délai de 30 jours, sauf si nous avons une obligation légale de les conserver plus longtemps.

8. Cookies et technologies similaires

L'application mobile utilise des technologies similaires aux cookies (tokens de session, préférences locales) pour améliorer votre expérience et maintenir votre session. Vous pouvez désactiver certaines de ces fonctionnalités dans les paramètres de votre appareil.

9. Services tiers

Notre application utilise les services suivants :

  • Supabase : Hébergement de la base de données
  • Firebase Cloud Messaging : Notifications push
  • Google Sign-In : Authentification

Ces services ont leurs propres politiques de confidentialité que nous vous encourageons à consulter.

10. Protection des mineurs

Notre application est destinée aux personnes âgées de 18 ans et plus. Nous ne collectons pas sciemment d'informations auprès de mineurs de moins de 18 ans.

11. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre. Les modifications importantes seront notifiées via l'application ou par email. La date de "dernière mise à jour" en haut de cette page sera modifiée en conséquence.

12. Contact

Pour toute question concernant cette politique de confidentialité ou vos données personnelles, vous pouvez nous contacter :

  • Email : ive.tawasol@gmail.com
